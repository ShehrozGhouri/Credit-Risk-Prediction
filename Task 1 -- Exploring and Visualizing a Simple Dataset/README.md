Task 1: Exploring and Visualizing a Simple Dataset
This project involves a basic exploratory data analysis (EDA) of the Iris Dataset.

🎯 Objective
* Load and summarize the dataset using **Pandas**.
* Identify relationships between features using **Scatter Plots**.
* Visualize data distributions using **Histograms**.
* Detect outliers using **Box Plots**.

📊 Tools Used
* **Python**
* **Seaborn**
* **Matplotlib**
* **Pandas**
