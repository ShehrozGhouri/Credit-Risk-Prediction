# Task 2: Credit Risk Prediction
This project involves building a Machine Learning model to predict whether a loan should be approved based on user data.

### 🎯 Objective
* Preprocess and clean a real-world financial dataset.
* Handle missing values and encode categorical data.
* Train a classification model to predict loan eligibility.

### 🧠 Model Details
* **Algorithm:** Logistic Regression
* **Accuracy Score:** 78.86%

### 📁 Folder Contents
* `Credit_Risk_Analysis.ipynb`: Full code and visualization.
* `Credit Risk Prediction.csv`: The dataset used for training.
* **Model/**: Contains the saved `.pkl` file for deployment.
