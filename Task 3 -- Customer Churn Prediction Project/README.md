# Task 3: Customer Churn Prediction
Predicting bank customer turnover using Machine Learning.

Objective
* Cleaned the Bank Churn dataset.
* Encoded categorical features like Gender and Geography.
* Trained a Random Forest Classifier.
* Analyzed Feature Importance to see why customers leave.

Model Performance
* **Accuracy:** 86%
* **Top Influencers:** Age, Estimated Salary, and Balance.
